Gesture Detection Extension

Gesture Detection Extension is a browser extension designed to enhance user interaction with web pages through hand gestures. Built with JavaScript and TensorFlow.js, this extension allows users to scroll web pages and perform other actions using hand gestures, making web navigation more intuitive.

The extension leverages machine learning to recognize and interpret various hand gestures in real-time. This technology is useful for users who prefer a more interactive and hands-free browsing experience.

Currently, there is no live link available for this project, but the code and implementation details can be shared upon request or through a future update.
