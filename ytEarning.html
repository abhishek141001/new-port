YouTube Earning Calculator

YouTube Earning Calculator is a tool designed to help users calculate YouTube video earnings based on a custom formula. Users can input a YouTube share link and compare metrics with other videos. The app also calculates earnings using a custom logic based on various factors.

The project is built using React for the frontend, with Redux for state management. The backend is powered by Node.js and Express, with data stored in MongoDB. The application leverages the YouTube API to fetch video data and perform calculations.

Features include the ability to analyze and compare video metrics, view detailed earning reports, and visualize data through interactive charts and graphs.
